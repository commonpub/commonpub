## Tips

- When reference docs and session logs conflict, **session logs are newer.**
- When docs and code conflict, **code is newer.** Flag the doc for update.
- `codebase-analysis/` is regenerated by spawning Explore agents over the code;
  treat it as a periodic dump, not a hand-curated narrative.
