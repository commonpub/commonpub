## Architecture as of session-close

```
/admin/layouts/[id].vue
  └── <AdminLayoutsToolbar>
  └── conflict-thrash banner (when editor.conflictThrashing)
  └── <AdminLayoutsAnnouncer>       ← singleton SR live region
  └── <DnDProvider previewTo="body" @click.self="clearSelection">
         ├── <AdminLayoutsCanvas onSelect selectedId>
         │     └── @click.self clears selection (3 levels: canvas / stage / zone-body)
         │     └── <LayoutSlot> (zone walker)
         │           └── <LayoutRow> v-for          ← makeDroppable + dispatchSectionDrop + Move Up/Down handler
         │                 └── <LayoutSection> v-for ← makeDraggable + selection click + Move Up/Down buttons
         │                       └── <component :is=registry.get(type).component> v-bind=propMap
         ├── <AdminLayoutsPalette>
         │     └── <AdminLayoutsPaletteTile> v-for   ← makeDraggable (palette-section-spec envelope)
         └── <AdminLayoutsInspector>
  └── <AdminLayoutsConflictModal>
```

**Drag flow** end-to-end:
1. Palette tile → row drop: `makeDraggable` (tile, payload `() => [0, [paletteDragPayload(def)]]`) → drag-over → row's `makeDroppable.onDrop` → `dispatchSectionDrop(event, row)` → branches `palette-section-spec` → `createSectionFromSpec(spec)` + `splice(insertAt, 0, newSection)` → row.sections mutated → editor's deep watcher → `useLayoutAutoSave` schedules debounced PUT through `editor.save()` single-flight guard.
2. Section within-row reorder: `makeDraggable` on section (payload `() => [0, [{kind:'section-instance', section, fromRowId}]]`) → drag-over different position in same row → dispatcher branches `section-instance` with `fromRowId===row.id` → splice-remove + splice-insert with target-adjustment for source-shifted-target → outcome `reordered`. Same save flow.
3. Move Up/Down keyboard path: button click → `LayoutRow.moveSection(section, direction)` → bounds-check (announce `narrateMoveBlocked` if at edge) → splice + announce `narrateReordered`. Same save flow.
4. Cross-row drop: dispatcher returns `noop 'cross-row-deferred-to-3b-B'` (silent — no narration; pointer feedback is enough).

## Audit findings

- R1-R4 sweep on the shipped work: **5 findings**

- Audit-of-audit: **2 findings**
  - A2-1: zone-body click-to-clear missing → third `@click.self` needed.
  - A2-2: min-height over-padded populated rows → gated to `:empty` pseudo-class.
