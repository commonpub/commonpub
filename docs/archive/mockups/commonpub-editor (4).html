§ 03 — Gradient Descent

The Learning
Rate Problem

Every neural network needs to find the bottom of a bowl — the lowest point of a loss function. It does this by shuffling weights in the direction that reduces error. But how big each shuffle is determines whether it finds the bottom or keeps bouncing around.

Body Text

This value — the learning rate — is one of the most consequential decisions in training. Set it too high, weights overshoot the minimum on every update. Set it too low, training barely moves.

Use the slider below to see how different learning rates affect a training run.

Body Text

Notice the sweet spot around 0.01–0.1? That's where the model makes meaningful progress without overshooting. In practice, researchers use adaptive optimizers like Adam that adjust the learning rate dynamically per parameter.
