Introduction

Insert block

The ESP32-S3 is Espressif's flagship microcontroller for AI and multimedia workloads. With dual Xtensa LX7 cores running at 240 MHz, 512 KB of SRAM, and hardware vector instructions for neural network inference, it's an ideal target for running quantized computer-vision models directly on the edge.

In this project we'll set up the camera pipeline using the ESP-IDF camera driver, load a int8 quantized MobileNet V2 SSD model from flash into PSRAM, and run inference at roughly 4–6 fps on a live video stream — classifying objects with over 80 COCO labels while staying entirely offline.

Flash the Firmware
Clone the repository and set up the ESP-IDF environment. Make sure you have ESP-IDF v5.1 or newer installed and activated in your shell. Connect the ESP32-S3 via USB-C and put it in download mode by holding the BOOT button while pressing RESET.
Terminal
$ git clone https://github.com/example/esp32s3-tinyml-vision
$ cd esp32s3-tinyml-vision
$ . $IDF_PATH/export.sh
# Configure target and build
$ idf.py set-target esp32s3
$ idf.py menuconfig # Enable SPIRAM, set camera pins
$ idf.py build flash monitor
I (532) camera: Camera init OK
I (614) tflite: Model loaded from flash — 3.2 MB
I (618) tflite: Allocating tensor arena: 1024 KB in PSRAM
Tip
If the build fails at the PSRAM allocation stage, open menuconfig → Component config → ESP PSRAM and ensure "Support for external, SPI-connected RAM" is enabled and the mode is set to Octal Mode PSRAM for the N16R8 variant.

// Initialize TFLite micro interpreter with PSRAM arena
static uint8_t tensor_arena[1024 * 1024] __attribute__((aligned(16)));
 
void run_inference(camera_fb_t *fb) {
  // Resize frame to model input: 300×300×3
  image_resize_linear(input_data, fb->buf,
    300, 300, 3, fb->width, fb->height);
 
  // Quantize float pixels → int8
  for (int i = 0; i < 300*300*3; i++) {
    input->data.int8[i] = (int8_t)(input_data[i] - 128);
  }
 
  TfLiteStatus status = interpreter->Invoke();
  if (status != kTfLiteOk) {
    ESP_LOGE("TFLITE", "Invoke failed: %d", status);
    return;
  }
 
  // Parse detection outputs (boxes, classes, scores, count)
  float *scores = interpreter->output(2)->data.f;
  int num_detections = (int)interpreter->output(3)->data.f[0];
 
  log_detections(scores, num_detections, 0.5f);
}
